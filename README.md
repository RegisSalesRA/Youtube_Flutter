# Flutter youtube api


## Getting Started

Before you start make sure you have get packages flutter
- use this command to get packages ( flutter packages get )

OBS: 
- You need create a API from this website https://developers.google.com/youtube/v3
- when you have genereted the key replace CHAVE_YOUTUBE_API with your key 
- CHAVE_YOUTUBE_API is located in lib/api.dart

## Now we are ready to run this project
- Make sure you have one device connected 
- Flutter doctor 
- Flutter run
